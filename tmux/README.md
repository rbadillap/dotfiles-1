# tmux cheatsheet

## Outside
`tmux new -s name`: Creates a new session called "name"
`tmux new -s name -n editor`: Creates a new session called "name" and names the first window "editor"

## Inside

Prefix is bound to `Ctrl`-`a`.

```
c  create window
,  name window
|  vertical split
-  horizontal split
x  kill pane
```
